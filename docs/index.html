How It Works

Built on the Karpathy LLM Wiki pattern.

1

Ingest

Add a YouTube URL. MCPTube extracts the transcript, detects scene changes, and describes key frames with a vision model.

2

Build Knowledge

An LLM extracts structured knowledge and writes wiki pages. Cross-references across videos are created automatically.

3

Ask & Reason

Query your knowledge base. The Ask Agent uses FTS5 search and LLM reasoning over compiled knowledge — not raw chunks.

v0.1 — Video Search

  • ❌ Chunk transcript, embed in vector DB
  • ❌ Find similar chunks per query
  • ❌ Re-search everything from scratch
  • ❌ Library of isolated videos

v2 — Knowledge Engine

  • ✅ LLM watches + reads, writes wiki pages
  • ✅ Agent reasons over compiled knowledge
  • ✅ Scene-change detection + vision model
  • ✅ Compounding knowledge base
